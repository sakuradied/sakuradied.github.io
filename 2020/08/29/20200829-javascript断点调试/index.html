20200829- javascript断点调试

使用Chrome浏览器F12进入开发者模式进行调试:

在于调试没太多可记录的和C的debugger几乎是差不多的可能就是快捷键上有些差距:
F9 下一步(逐条)
F10 跨步 略过函数执行
F11 步入 依次到函数(等待执行,进入函数)
shift+F11 步出 跳到函数末端
F8 继续执行(直到下一个断点或者程序结束)

这边应该可以理解为寄存器区(区别还是挺大的)

watch 显示任意表达式的值
call stack 显示嵌套的调用链
scope 显示当前的变量(local 局部变量 global全局变量)
